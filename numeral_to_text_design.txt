Design a recursive function to convert numerals to text in Nahuatl (Nauatl).

Nahuatl number names derive from a base 20 numeral system.

The function should find the greatest digit position (base 20) of the numeral, name it, and then recursively call the function with the remainder.

Example 421(base 10) = 111(base 20)
400 - sentsontli
20 - sempoalli
1 - se

sentsontli iuan sempoalli iuan se
